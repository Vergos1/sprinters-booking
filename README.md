## About

Sprinters Booking is a web application for booking bus tickets and tracking parcels on Ukraine–Europe routes. The app allows passengers to quickly book a seat online and track the real-time location of their bus or parcel.

🔗 **Live:** [sprinters](https://sprinters-blond.vercel.app)

## Features

- **Online Booking** — fast ticket booking without phone calls, choose passenger or carrier mode
- **Driver Location** — track the real-time location of your bus or parcel
- **Trip Status** — get notified when the driver departs
- **Parcel Tracking** — track parcels sent to family or friends in real time
- **Route Search** — browse available Europe-bound routes with date selection

## Getting Started

```bash
# Install dependencies
yarn install

# Start development server
yarn dev

# Build for production
yarn build

# Preview production build
yarn preview
```
